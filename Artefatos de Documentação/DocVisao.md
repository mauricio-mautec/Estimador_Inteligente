# Documento de Visão
## Objetivo
- Desenvolver uma plataforma para apoiar a produção de uma padaria onde o usuário recebe uma previsão da produção diária de produto específico.
## Escopo
- O sistema deverá treinar um modelo de IA para predizer a quantidade a ser produzida de um produto com base em uma série temporal com o resultado correto da produção anterior.
- A interface de usuário terá duas frentes: uma web, responsiva e outra via interação com o aplicativo de mensagens Telegram.
  - Na interface web, o usuário poderá submeter dados de treinamento e escolher o produto a ser modelado.
  - Poderá ainda submeter o valor da produção do dia anterior e solicitar a previsão da produção atual.
  - A interface vai Telegram apoia somente a utilização do modelo de predição, nos mesmos moldes da interface web.
